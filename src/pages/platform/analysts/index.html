Support for all TM1 Excel functions including DBRW, DBR, DBRA, and more — with full hierarchy support across all formula types.

Dynamic columns & cross-drill

Create reports with dynamic column structures that automatically expand with your hierarchy. Cross-drill from summary to detail without leaving your workbook.

IBM Planning Analytics data in Microsoft Power BI

PowerConnect provides a live, secure connection from your TM1 cubes to Power BI — so your dashboards always reflect current data without manual exports.

Live TM1 connection

Connect Power BI Desktop and Power BI Service directly to your IBM Planning Analytics cubes — on-premises, cloud, or hybrid — for always-current reports.

Automatic dimension sync

IBM Planning Analytics hierarchies and dimensions sync automatically to Power BI. No manual updates when your model changes — reports always reflect the latest structure.

Security integration

PowerConnect integrates with your existing IBM Planning Analytics security model — so users only see the data they're entitled to, both in TM1 and in Power BI.

Ad-hoc & scheduled refresh

Refresh data on demand or on a schedule. Works with Power BI Desktop, Service, and mobile — without any manual data exports or file transfers.

Rich Power BI visuals

Use the full Power BI visual library on top of TM1 data — charts, maps, slicers, and custom visuals — without flattening cubes or losing hierarchies.

On-prem, cloud or hybrid

Whether your TM1 estate runs on-premises, in IBM Cloud, or in a hybrid setup, PowerConnect connects the same way — no separate flavours to maintain.
