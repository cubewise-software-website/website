Write better TM1 code, faster

Arc is the IDE TM1 developers have wanted for decades. Every feature is designed to eliminate friction and help you build with confidence.

Full-featured TI editor

Syntax highlighting, intelligent autocomplete, code folding, and find-and-replace across your entire model — not just one tab at a time.

Debugging & breakpoints

Step through TI code line by line, inspect variable values in real time, and pause at any condition. No more logging to text files.

Code templates & snippets

Access hundreds of reusable, approved code templates. Create your own snippets and share them across your team to standardise the codebase.

Multi-server management

Connect to multiple TM1 instances simultaneously. Switch between environments, copy objects, and manage settings — all in one window.

Rules editor

Edit cube rules with full syntax highlighting, section folding, and autocomplete for dimension and element names — a huge improvement over the default editor.

Security & administration

Manage users, groups, and object security in one screen. Update server settings on the fly and access logs without switching contexts.

Apply DevOps discipline to your TM1 team

Arc+ adds the rigour of modern software development to IBM Planning Analytics — so your team ships faster, with less risk.

Impact analysis

Know exactly what will be affected before making a change. Visualise dependencies across your entire TM1 model and reduce the risk of unintended consequences.

Automated unit testing

Write tests once as part of your development phase. Run test collections in sequence and use code coverage analysis to see exactly what's being tested.

Best practice enforcement

Define rules for how TM1 objects should be built and Arc+ enforces them automatically — keeping your codebase clean and consistent across the whole team.

Deployment packages

Arc+ automatically resolves all object dependencies and bundles them into a deployment package. Deploy to production live without bringing services down.

Migration logs & rollback

A full deployment history with downloadable migration logs. If something goes wrong, roll back changes quickly and confidently.

To Do list & backlog

Manage your development backlog directly inside Arc+. Keep track of what needs building, what's in progress, and what's ready to deploy.
