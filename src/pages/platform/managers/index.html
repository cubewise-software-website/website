Know what is happening — always

Pulse gives you real-time visibility across every server, a complete audit trail of every change, and the confidence to answer platform questions without asking your team.

Real-time environment health

24/7 monitoring across every TM1 server with instant alerts for user locks, chore failures, memory thresholds, and model changes — delivered by email or integrated into your alerting platform.

Complete change audit trail

A searchable history of every user action, object change and chore execution. Answer "what changed, when, and who changed it?" in seconds — with evidence for auditors when they ask.

Controlled live migration

Deploy changes to production without maintenance windows. Full deployment history, downloadable logs, and one-click rollback — every release is a controlled event.

Performance and usage insight

Capacity planning data

Trend memory, CPU, and concurrent users over time across every server. Plan upgrades and infrastructure spend with hard numbers, not gut feel.

Stakeholder reporting

Approve changes with confidence

Arc+ brings DevOps discipline to IBM Planning Analytics — impact analysis before every change, automated tests, enforced standards and one-click rollback. Risk becomes measurable, not a guessing game.

Impact analysis

Visualise exactly what will be affected by any proposed change across the entire TM1 model. Approve or reject with full evidence before anything reaches production.

Best-practice enforcement

Define the rules for how TM1 objects should be built. Arc+ enforces them automatically — keeping the codebase clean, consistent and auditable across the whole team.

Automated testing

Unit tests that run on every change, with code coverage analysis. Regressions are caught during development — not after the month-end report goes wrong.

Separation of duties

Keep development and production operations separate, with approval workflows for every deployment. Managers stay in control of what changes in production — and can prove it.

Controlled deployments

Bundle changes into reviewable deployment packages with all dependencies resolved. Approve, schedule and deploy as one auditable unit — no surprise side-effects.

Audit-ready evidence

Every approval, test run, and deployment is logged automatically. Hand auditors a documented trail instead of reconstructing it from memory at year-end.
