Smart Support & Monitoring

Get ahead of bugs and bottlenecks with advanced monitoring built into your development environment.

Safeguard Critical Objects

Lock critical TM1 objects to prevent accidental changes. Arc+ lets you protect production rules, processes, and dimensions so developers can't overwrite them without explicit authorisation.

Advanced Monitoring

Monitor your TM1 server instances from within Arc via the upgraded Sessions Page. See active connections, running threads, and server health without leaving your development environment.

Pulse Alerts in Arc

On top of receiving email alerts, Arc users will see a pop-up notification each time a Pulse alert is triggered — so you're always informed of issues as they happen.

Change Tracking

Full audit trail of every change made to your TM1 objects, recorded by user and timestamp. Understand who changed what and when — directly from within Arc.

Chore & Process History

View Pulse chore and process history data directly in the Arc object editors. Quickly see run durations, execution counts, and any issues in past runs.

Object Usage Tracking

Track TM1 object usage — processes, rules, subsets, cube views, and data changes — by sending activity data to Pulse. Visualise it in Pulse Explorer with the built-in User Objects Usage dashboard.

Model Search

Access Pulse Model Search from within Arc. Search across your entire TM1 model — rules, processes, dimensions, and more — to find objects and references instantly.

Development & Migration

Faster build cycles, safer migrations, and deeper insight into your TM1 models.

Flow Diagram

From a cube view, process editor, or rule editor, generate the Pulse flow diagram for the selected object with a single click. See all object relationships without reading the code.

Blame Report

See exactly who wrote each line of a TM1 process or rule, and when. The Blame Report makes it easy to trace the origin of any piece of code in your model.

Side-by-side Comparison

Compare two versions of any TM1 object side by side. Instantly see what changed between versions before applying a modification.

Compare Dimensions

Find differences between two dimensions or hierarchies — even across different server instances. Quickly identify missing elements or elements with different parents.

Migrate TM1 Objects

Select multiple TM1 objects and migrate them all at once using Pulse — directly from Arc. Pulse automatically finds all dependencies, reducing the risk of missing objects during a migration.

Test Hub

Automated unit testing and test-driven development for TM1 models. Set up individual unit tests during development and group them into collections to run in sequence.

Best Practice Validation

Predefined rules validate naming conventions and flag common pitfalls in cube rules and TurboIntegrator processes — giving you pre-emptive visibility of issues before they become problems.
