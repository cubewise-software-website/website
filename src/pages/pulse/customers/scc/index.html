How SCC migrates TM1 packages live with zero downtime

"Pulse gives us as developers the option to migrate packages live without the need to bring down services."

About SCC

SCC is Europe's largest independent technology solutions company, providing European businesses and public sector organisations with IT infrastructure, cloud, and managed services. Their TM1 model is core to the business and downtime needs to be kept to an absolute minimum — especially during working hours. The model connects to multiple data sources, meaning any stop and restart of services risks disrupting data connectivity.

The Challenge

When migrating manually-created packages from development to production, SCC had to schedule downtime — a significant business disruption. There was also a high risk that objects could be missed during manual migrations, requiring the process to be repeated. With a model core to business operations, any period of unavailability had direct impact on users and data flows.

The Solution

Pulse's migration functionality transformed SCC's deployment process. Rather than scheduling maintenance windows, developers can now migrate packages live while services remain running. Pulse automatically scans the model and attaches all dependencies to a migration package — so if you select a cube, Pulse automatically includes the dimensions and rules that build it, eliminating the risk of missing objects.

When an object already exists in the target environment, Pulse only migrates the changes and differences — ensuring nothing that already exists is affected unnecessarily.

The Results

  • Zero scheduled downtime for migrations — the system remains available 24/7
  • Automatic dependency detection removes the risk of missing objects during migration
  • Only changed objects are migrated, protecting existing configurations in the target environment
  • Time-consuming manual package creation eliminated, freeing developers for higher-value work
  • Faster handover of completed models to users without waiting for maintenance windows
