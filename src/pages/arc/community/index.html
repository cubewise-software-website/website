Community Edition

Cube Viewer

Support for hierarchies, MDX and native views, public and private views, snapshot, slice and active forms, search bar and more.

Subset Editor

All the features you find in Architect, plus a search box to quickly find elements and a new way to visualise and edit MDX queries.

History & Favourites

Quickly access the latest objects you worked on in the History tab. Pin current development priorities to Favourites to stay focused and resume your workflow after an interruption.

Desktop Application

Arc is available as a desktop application on Windows. The main advantage over Arc Server is the ability to use multiple windows simultaneously.

Summary Views

Arc includes summary views that organise your model's dimensions, cubes, processes and chores at a glance.

To Do List

Create task lists with steps and actions. Each action can open a TM1 object or execute a process or chore directly from Arc.
