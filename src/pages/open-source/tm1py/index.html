Free Python framework for IBM Planning Analytics

TM1py

Programmatic control over TM1 — automate, integrate, and extend your Planning Analytics applications with Python.

View on GitHub Explore Samples

Connect TM1 to anything

TM1py bridges IBM Planning Analytics and Python, giving you complete programmatic control over every object and setting in TM1. Combined with Python’s vast library ecosystem, the integration possibilities are unlimited.

  • Create seamless integrations between TM1 and cloud systems
  • Infuse AI and forecasting algorithms into your planning cycle
  • Synchronise hierarchies and data across multiple TM1 instances
  • Export data in any format — CSV, XLSX, XML, JSON, and more

Automate what TM1 can’t reach natively

Turbo Integrator has limits. TM1py doesn’t. Automate tasks that would otherwise require manual intervention, and update objects that TI cannot touch.

  • Automate manual and error-prone TM1 tasks
  • Update chores, private objects, and Excel applications programmatically
  • Execute data integrity and security model checks automatically
  • Run processes in parallel with a single connection using Rushti

70+ ready-to-use samples

The fastest way to get started is to download the TM1py samples repository from GitHub. Each sample is a self-contained script demonstrating a real-world use case.

  • Getting started scripts for common TM1 operations
  • Data load and export examples
  • Security and administration automation
  • MDXPy — generate MDX queries programmatically without writing raw MDX

Community powered

TM1py is free and open source. The source code is hosted on GitHub — contributions, bug reports, and pull requests are welcome from the TM1 community.

  • Free to use with no licensing restrictions
  • Actively maintained by Cubewise and community contributors
  • Powers other open source projects including Rushti, CubeCal, MDXPy, and OptimusPy
  • All code provided without warranty — use and modify freely
