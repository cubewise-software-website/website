Our Story

Built by TM1 experts,
for TM1 teams

We build software that helps IBM Planning Analytics / TM1 teams work faster, smarter, and with more confidence — informed by deep expertise and close collaboration with the people who use it every day.

"Cubewise didn't set out to build software. We set out to solve problems."

IBM Planning Analytics / TM1 is one of the most powerful planning platforms in the world — built for calculation speed, deep Excel integration, and data volumes that other platforms struggle to match. Cubewise was founded by experts who understood that power deeply, and who set out to make it even more accessible, productive, and professional for every team that relies on it.

We listened to developers, administrators, and finance teams — and built software that amplifies what they can do with IBM Planning Analytics / TM1. The result is a platform of six specialised software, each one grounded in real-world experience and designed to work together seamlessly.

The foundation was laid even earlier, with Bedrock — a living library of TI code templates, best practices, and toolkits distilled from years of project work across industries. Bedrock reduced TI coding effort by up to 50% and established the standards on which everything else was built.

Solving real life challenges for IBM Planning Analytics teams.

Arc

TM1 developers deserved a professional IDE — not a basic editor that slowed them down.

Arc is the professional IDE for IBM Planning Analytics / TM1, with smart search, templates, blueprints, and a lightweight install. It gives developers a modern environment that makes development faster, more precise, and more enjoyable.

Developers work faster, with greater precision and unified coding practices.

Arc+

TM1 teams were ready for DevOps — but had no platform for automated testing, impact analysis, and structured deployment.

Arc+ brings DevOps to IBM Planning Analytics / TM1 — automated testing, impact analysis, and deployment pipelines in one platform. Teams can build, test, and deploy with the same rigour as any modern software project.

TM1 teams adopted modern software practices without leaving their ecosystem.

Pulse

Administrators had limited visibility inside their TM1 environments — making monitoring, auditing, and migration unnecessarily risky.

Pulse gives administrators complete visibility and control over their IBM Planning Analytics environment. Real-time monitoring, alerting, migration tooling, and full audit trails — in one place.

Support calls dropped. User locks resolved proactively. Migrations de-risked.

Slice

Finance teams needed a modern Excel experience for IBM Planning Analytics — with hierarchies, cloud connectivity, and REST API support.

Slice is the modern Excel add-in for IBM Planning Analytics, with Active Forms, writeback, cross-drill, dynamic columns, and direct REST API connectivity. It honours Excel familiarity and adds the capabilities teams have always needed.

Smooth transition from Perspectives, with richer reporting and built-in auditing.

PowerConnect

Business intelligence teams wanted TM1 data in Power BI — but there was no reliable, secure way to connect the two platforms.

PowerConnect brings IBM Planning Analytics data directly into Microsoft Power BI — with seamless synchronisation of data, metadata, and security. Executives and analysts can build live, interactive dashboards without manual exports.

Executives gained real-time visibility into planning data with scheduled refreshes.

Atmosphere

Integrating TM1 with cloud data sources required complex middleware, long delivery cycles, and ongoing maintenance.

Atmosphere connects IBM Planning Analytics / TM1 to the cloud with speed, security, and simplicity. With 34+ pre-built connectors, real-time data capture, and an intuitive admin portal — teams focus on planning, not plumbing.

Integration became seamless, secure, and focused on outcomes rather than plumbing.

TM1py

Python was transforming data science and automation — but TM1 had no way to participate in that ecosystem.

TM1py bridges IBM Planning Analytics / TM1 and Python, unlocking integrations with Salesforce, SAP, Google Sheets, and more. It opens the door to machine learning, forecasting, and anomaly detection on planning data.

TM1 became programmable, flexible, and future-ready.

Canvas & Apliqo UX

Customers needed rich, branded web interfaces on top of TM1 — but building them required deep developer expertise and significant effort.

Canvas is a developer framework for building custom web frontends via the TM1 REST API. Cubewise partnered with Apliqo to evolve it into Apliqo UX — a no-code platform for rich web applications on top of IBM Planning Analytics.

From code to no-code — TM1 became accessible to a broader audience.
